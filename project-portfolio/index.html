Week 1 - 13 Remote Sensing Change Detection
Project Summary

Remote Sensing of the Environment

This project focuses on remote sensing-based environmental monitoring.

We aim to create a change detection dataset using recent Sri Lankan Cyclone Dithwa satellite imagery.

The project also explores algorithmic improvements to accurately detect and analyze environmental changes from multi-temporal remote sensing data.

The study emphasizes careful preprocessing, consistent annotation, and quantitative evaluation to ensure academic quality outcomes.

Key Focus Areas

  • Study Context: Cyclone Dithwa impacts in Sri Lanka using pre- and post-event satellite imagery.
  • Data Preparation: Co-registration, atmospheric correction, cloud masking, and labeling guidelines.
  • Change Detection: Compare classical and learning-based methods with robust accuracy metrics.
  • Deliverables: Curated dataset, analysis report, and visual change maps for review.
